About UpLearn


The original UpLearn application was a pure Rails app that I built earlier this year. I'm rebuilding that app here, using AngularJS on the front-end and a Rails API as the back-end.

The code for this app is here on GitHub.

Early days so not at feature parity yet. As I build the app, I'm keeping track of the known bugs and quirks here:


  • Search results on home page always has title "Latest" becuase of the $first variable with ng-repeat
  • Cannot get bootstrap glyphicons to play ball with the Rails asset pipeline
  • Upvote/downvote buttons not fully functional - need to tie to a user in database
  • Highlighted active link in menu is not dynamic, so always highlights "Home" link
  • When nobody logged in, system throws an error becuase of empty user {}
  • Bootstrap Nav Bar disappears below 767px width
  • Logout on homepage and then immediately trying to log back in throws this error: "422 (Unprocessable Entity)". Logging out from other pages does not cause this issue.
